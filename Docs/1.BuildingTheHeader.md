## Adding Basic Styles

```css
/*
this is called a universal selector
*/
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
/*
using added google fint
line height 1.7 means 1.7 times referance size

*/
body {
    font-family: "Lato", sans-serif;
    font-weight: 400;
    font-size: 16px;
    line-height: 1.7;
    color: #777;
    padding: 30px;
}

.header {
    height: 95vh;
    background-image: linear-gradient(to right bottom, rgba(126, 213, 111, 0.8), rgba(40, 180, 133, 0.8)), url("../img/travel.jpg");
    background-size: cover;
    background-position: top;
    background-repeat: no-repeat;
    clip-path: polygon(0 0, 100% 0, 100% 80vh, 0 101%);
}
```

```html
    <link rel="shortcut icon" href="./img/favicon.png" type="image/x-icon">
    <title>Travel</title>
</head>
<body>
<header class="header">
    <div class="logo-box">
        <img src="./img/logo-white.png" alt="logo" class="logo">
    </div>
    <div class="text-box">
        <h1 class="heading-primary">

```

```css
/*
```
